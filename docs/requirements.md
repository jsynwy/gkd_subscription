# 订阅规则开发需求文档

## 项目概述

为本 GKD 订阅模板项目新增 5 个应用的跳过广告规则。

## 涵盖应用及包名

| 应用名称 | 包名 |
|---------|------|
| 小红书 | `com.xingin.xhs` |
| 哔哩哔哩 | `tv.danmaku.bili` |
| 高德地图 | `com.autonavi.minimap` |
| 酷安 | `com.coolapk.market` |
| 萤石云视频 | `com.videogo` |

## 优先级策略设计

每个应用的规则结构：

| 优先级 | 规则类型 | 说明 |
|--------|---------|------|
| 10 | 开屏广告 | 应用启动时立即处理，匹配开屏页跳过按钮 |
| 6 | 弹窗广告 | 应用内弹窗，匹配弹窗广告关闭按钮 |

## 各应用规则详情

### 1. 小红书 (com.xingin.xhs)

- **开屏广告** (priority: 10)
  - 匹配开屏页跳过按钮
- **弹窗广告** (priority: 6)
  - 匹配首页/个人页弹窗广告关闭按钮

### 2. 哔哩哔哩 (tv.danmaku.bili)

- **开屏广告** (priority: 10)
  - 匹配启动页跳过按钮
- **弹窗广告** (priority: 6)
  - 匹配首页活动弹窗关闭按钮

### 3. 高德地图 (com.autonavi.minimap)

- **开屏广告** (priority: 10)
  - 匹配启动页跳过按钮
- **弹窗广告** (priority: 6)
  - 匹配地图页弹窗广告关闭按钮

### 4. 酷安 (com.coolapk.market)

- **开屏广告** (priority: 10)
  - 匹配启动页跳过按钮
- **弹窗广告** (priority: 6)
  - 匹配应用详情页/首页弹窗关闭按钮

### 5. 萤石云视频 (com.videogo)

- **开屏广告** (priority: 10)
  - 匹配启动页跳过按钮
- **弹窗广告** (priority: 6)
  - 匹配监控页/首页弹窗广告关闭按钮

## 验收标准

1. 5 个应用的规则文件全部创建完成
2. 每个应用包含开屏广告和弹窗广告两种规则
3. 优先级设置正确（开屏 10，弹窗 6）
4. 所有规则通过 `pnpm run check` 验证
5. 成功构建输出 `dist/gkd.json5`
