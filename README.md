# GKD 订阅

适用于 [GKD (搞快点)](https://gkd.li/) 的第三方订阅规则，自动跳过开屏广告和弹窗广告。

## 订阅地址

```
https://raw.githubusercontent.com/jsynwy/gkd_subscription/main/dist/gkd.json5
```

## 使用方法

1. 安装 GKD 应用
2. 打开 GKD → 订阅 → 添加订阅
3. 粘贴上方订阅地址
4. 开启无障碍服务权限
